# Homebrew Tap

Custom Homebrew tap for Stefan's tools.

## Installation

```bash
brew tap stefanlegg/tap
```

## Available Casks

### Scry 🔮

A lightweight macOS menu bar app that shows your running dev servers at a glance.

```bash
brew install --cask stefanlegg/tap/scry
```

Or after tapping:

```bash
brew install --cask scry
```
